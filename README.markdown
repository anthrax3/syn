# syn

## man syn

```
syn(1)                    BSD General Commands Manual                   syn(1)

NAME
     syn -- Syntax control for the command line

SYNOPSIS
     syn [-nvaANdpPciCIohl]

DESCRIPTION
     syn uses OS X's natural language processing tools to tokenize and high-
     light text (from standard input) that match specified tags (passed in as
     flags).

     Tag-setting flags:

     -a       Match adjectives

     -A       Match adverbs

     -c       Match conjunctions

     -C       Match classifiers

     -d       Match determiners

     -C       Match classifiers

     -H       Match personal (human) names

     -i       Match interjections

     -I       Match idioms

     -l       Match place names (locations)

     -n       Match nouns

     -N       Match pronouns

     -o       Match "other" words

     -p       Match particles

     -P       Match prepositions

     -V       Match verbs

     Other flags:

     -v       Prints the current version

     -h       Prints a message decribing usage

Darwin                         December 24, 2013                        Darwin
```
